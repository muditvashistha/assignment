
Please run npm install first in order to install the required dependencies for the project to be functional!

In the project directory, you can run:

### `npm start`

EMPLOYEE DIREECTORY

The project is a searchable list of employees which have a ROLE and DEPARTMENT associated to them. The project showcases the following functionalities:-

1) List employees (name, role, department)
2) Search by name/department
3) Add/edit employee details
4) Responsive design
